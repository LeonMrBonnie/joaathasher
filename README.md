## Joaat (Jenkins-One-At-A-Time) Hasher CLI Tool

# How to install

```
npm install -g joaathasher
```

# How to use

```
joaat <hash> <?file>
```

**Example output:**

Without creating a file:
```
> joaat test123
> -1001436764
```

With creating a file:
```
> joaat test123 hash.txt
> -1001436764
```
